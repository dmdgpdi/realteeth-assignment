# Agent Rules

## 1. 언어 규칙 (Language Rules)
- **모든 대화와 답변은 반드시 한국어로 진행합니다.**
- 코드 내의 주석이나 커밋 메시지는 프로젝트 컨벤션에 따르되, 설명은 한국어로 제공합니다.

## 2. 규칙 참조 (Rule References)
- `agents/` 폴더 내에 정의된 모든 규칙 파일들을 읽고 준수해야 합니다.
- 새로운 작업을 시작하기 전에 관련 도메인이나 기술 스택에 해당하는 `agents/*.md` 파일이 있는지 확인하세요.
```
// example
- Follow architecture rules in agent.architecture.md
- Follow frontend rules in agent.frontend.md
- Follow domain rules in agent.weather-domain.md
- Follow UI rules in agent.ui.md
```
